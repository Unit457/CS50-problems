Esse projeto é um "assistente de utilidades".
Ao iniciar o programa, o usuário recebe as opções:
- Calculador de média
- Conversor de unidades
- Simulador de caixa eletrônico
- Sair

O simulador de caixa eletrônico foi o mais complicado de fazer.
Esse projeto me ajudou a colocar em prática e exercitar os conteúdos que aprendi na Semana 1 ("Correctness, Design, Style. Header Files. Libraries. Types. Conditionals. Variables. Loops. Constants. Comments. Pseudocode. Operators.") ao mesmo tempo que foi uma introdução para o tratamento de erros de serviços.
