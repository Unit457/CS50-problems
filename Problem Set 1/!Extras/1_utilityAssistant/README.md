Esse projeto é um "assistente de utilidades".
Ao iniciar o programa, o usuário recebe as opções:
- Calculador de média
- Conversor de unidades
- Simulador de caixa eletrônico
- Sair

O simulador de caixa eletrônico foi o mais complicado de fazer.
Primeiro exercício autoral em C, então me ajudou a fixar muitas coisas que foram estudadas previamente, ao mesmo tempo que foi uma introdução para o tratamento de erros de serviços e criação de menus no terminal.
